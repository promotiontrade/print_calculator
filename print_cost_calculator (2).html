Все расчёты выполняются локально в браузере. Для DTF себестоимость = стоимость изделия + (прайс поставщика по формату и тиражу [+ припекание] × (1 + наценка поставщика) + плата за прижим). Для остальных видов печати: (подготовка + трафарет×цвета)/тираж + краска×цвета + материал×см² + базовая ставка. Отредактируйте расценки в соответствующих разделах настроек.
